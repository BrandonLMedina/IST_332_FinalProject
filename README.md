# IST_332_FinalProject

Brandon Medina Final Project for IST 332 NLP @ CGU

Google Colab Link: [Collab Notebook](https://colab.research.google.com/drive/1QdV6y5OYibCY03V157N0DwoDw9gvrLA0?usp=sharing)

## Project Overview

This project explores real-time event recognition within Major League Baseball social media streams.

## Project Directory

- [`nlp_final_project.py`](nlp_final_project.py) – standalone Python script that runs the full data ingestion, preprocessing, model training, and evaluation pipeline from the command line.
- [`NLP_Final_Project.ipynb`](NLP_Final_Project.ipynb) – interactive notebook mirroring the script while exposing intermediate outputs, charts, and exploratory analyses.
- [`Brandon Medina Real-Time Event Recognition in MLB Social Media Streams.pdf`](Brandon%20Medina%20Real-Time%20Event%20Recognition%20in%20MLB%20Social%20Media%20Streams.pdf) – written report detailing the methodology, experiments, and key findings from the final submission.
- [`Real-Time Event Recognition Using MLB.pptx`](Real-Time%20Event%20Recognition%20Using%20MLB.pptx) – presentation deck summarizing objectives, design, and results.
